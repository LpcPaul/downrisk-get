## 四、PEG估值法核心洞察

### PEG估值框架
```
PEG = PE / 利润增速%

PEG < 0.5: 严重低估 → 调整+5%
PEG 0.5-0.8: 合理偏低 → 调整+2%至+3%
PEG 0.8-1.2: 合理 → 调整0%
PEG 1.2-1.5: 合理偏高 → 调整-2%至-3%
PEG > 1.5: 高估 → 调整-5%至-8%
利润负增长 + PE>30: 严重高估 → 调整-8%至-12%
```
