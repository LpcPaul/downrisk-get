## 🔬 方法论说明

### 风险计算 (Scheme C)
```
损失风险 = 0.6 × σ下行 + 0.3 × (β × σ总) + 0.1 × 脆弱度
```

- **σ下行**: 真实下行波动率（日频半方差年化）
- **σ总**: 总波动率
- **β**: 个股相对行业敏感度
- **脆弱度**: 结构性风险加点（融资/监管/客户集中等）

### 期望收益计算
```
ER = (目标倍数/当前倍数 × (1+增长率) - 1) + 股息 + 回购
ER执行 = ER原始 × 执行概率
```

### 值博率
```
值博率 = 期望收益 ÷ 损失风险
```
